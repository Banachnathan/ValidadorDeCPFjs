CPF Validator

Meu primeiro projeto utilizando JavaScript.

Este projeto foi desenvolvido com o objetivo de praticar lógica de programação, manipulação de strings e eventos no JavaScript, criando um validador de CPF funcional.

Funcionalidades:
Remove caracteres não numéricos do CPF
Detecta CPFs inválidos com sequência (ex: 11111111111)
Calcula os dois dígitos verificadores
Informa se o CPF é válido ou inválido na interface
🛠 Tecnologias utilizadas
HTML
CSS
JavaScript

O que aprendi:
Manipulação de strings com JavaScript
Estruturas de repetição (for)
Lógica de validação (cálculo de dígitos verificadores)
Uso de eventos no DOM (addEventListener)
Organização básica de código

Objetivo:
Esse projeto faz parte do meu processo de aprendizado em programação.
A ideia é evoluir progressivamente, criando projetos cada vez mais completos.

Como usar:
Abra o arquivo index.html no navegador
Digite um CPF no campo
Clique no botão para validar

Observação:
Este é um projeto inicial e faz parte da minha evolução como desenvolvedor.
